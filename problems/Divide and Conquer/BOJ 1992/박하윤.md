- 내가 이해한 문제내용
	- N x N크기의 영상속 0과 1을 입력받음
	- 0으로만 혹은 1으로만 이루어져 있을 때 0 혹은 1을 압축하여 출력
	- 혼합되어 있을 경우엔 왼쪽 위, 오른쪽 위, 오른쪽 위, 오른쪽 아래 순으로 4개로 분활하여 다시 찾음(반복)

- 접근 방식
	- 무엇을 출력하는지 이해하기 위해 그림 그림
	- 정복 분활 관련 공부
	- 재귀함수를 통해 표현 

- 어려웠던 점
	- 처음 숫자와 다음 숫자가 다른지 같은지 확인하는 for문의 알고리즘 이해하는데 시간이 오래걸림

- 복잡도
	- 시간복잡도: O(NlogN)
	- 공간복잡도: N^2
