- 접근 방식
	- 첫번째 접근
	- N combination C개의 행렬생성
	- 순열을 전부 집어 넣는다.
	- 순열의 차이중 최소 값을 끝열에 넣는다.
	- 서로 비교하여 최소값 찾기
	- 두번째 접근
	- 파라메트릭 서치
	- 최적화 문제 : 가장 인접한 두공유기 사이의 최대거리
	- 결정 문제 : 가장인접한 두 공유기
	- 결정문제를 만족하는 값중에서 최소값을 찾는다

- 어려웠던 점
	- 파라메트릭 서치
	- 어레이의 효율적인 생성과 접

	-  시간복잡도:  O(nlogn)
	-  공간복잡도:  O(n)
